   -$ git branch branch2  - со ова команда се креира гранка branch2

24.-$ git merge branch2 - со оваа команда сакав да ги спојам branch2  со master гранката, но се јаве конфликт.
Бидејќи конфликтот шо ми се јавуват со порака:
Auto-merging numbers.txt
CONFLICT (content): Merge conflict in numbers.txt
Automatic merge failed; fix conflicts and then commit the result.
Одлучив да го решам конфликтот, променувајќи со веќе познати комади, а тие се:
   -$ nano numbers.txt
   -$ git add numbers.txt
   -$ git commit -m "Conflict"

25.-$ git push -u origin master - тука правилно последна синхронизација со GitHub, и све што сме правеле 
на локално репо префрламе таму.
